# **🌈 REPL 사용하기**

자바스크립트는 스크립트 언어라서 즉석에서 코드를 실행할 수 있음

- `REPL`이라는 콘솔 제공

- R(Read), E(Evaluate), P(Print), L(Loop)

- 윈도에서는 명령 프롬프트, 맥이나 리눅스에서는 터미널에 node 입력

![REPL](https://github.com/leekoby/leekoby.github.io/assets/118284808/8b7f0ec1-985f-47fa-ab96-83ce22ba1a2c){: width="500" height="500" }

프롬프트가 `>` 모양으로 바뀌면, 자바스크립트 코드 입력
입력한 값의 결괏값이 바로 출력됨.

- 간단한 코드를 테스트하는 용도로 적합

- 긴 코드를 입력하기에는 부적합

![NODE 프롬프트](https://github.com/leekoby/leekoby.github.io/assets/118284808/7f104a96-0f84-4ab6-b95b-e7d3e639fb9f){: width="500" height="500" }
